Popular Modding Tools Tutorials

 

 

Intent

 

   The SSE TLC is a comprehensive collection of tutorial links from all corners of the internet, sorted by subject, to help modders in developing new skillsets. It was felt that this material would be better found and utilized, especially by those with no technical background, by hosting a list of available links in one place. Many SSE tools are interconnected and users should have basic understanding in many to become proficient modders able to make their own stable playlists of mods, but sometimes lack even the basic knowledge to know what questions to ask.

 

   Each tutorial is identified as being beginner, intermediate, or advanced. Following each tutorial title and link is brief informational content with the author's name, summary, format, and date of creation. Links to explicitly pay-to-access material are not currently listed, though several of the sites linked to offer subscriptions for reduced ads or host authors who have private Patreon accounts.

 

   Here is an explanation of the difficulty identifier icons utilized within the TLC:

 

•  {% include image-inline.html max-width="20px" file="img/SqOC.png" alt="OC" %} This designates "official content". This material has been authored by the developer(s). Always read these first in any subject unit as the developers know their tools best. Other user videos will clarify or expand on this material.

 

•  {% include image-inline.html max-width="20px" file="img/Sqbegin.png" alt="B" %} This designates "beginner" content. Most users will be able to understand this material without any prerequisites.

 

•  {% include image-inline.html max-width="20px" file="img/Sqinter.png" alt="I" %} This designates "intermediate" content. Typically, new users will need to review and understand beginner content before attempting to master this material.

 

•  {% include image-inline.html max-width="20px" file="img/Sqadvan.png" alt="A" %} This designates "advanced" content. Do not attempt to use material covered in these tutorials unless you have a broad, in-depth understanding of the tool you are attempting to use.
