== Intent
When I looked for some old Morrowind mods it took me a bit to find them on alternate sites. I have also found forum posts that provided users with usefull information. However, you will find that most of those posts were not maintained or the sites are no longer available. With none of the links really working the information isn't as usefull. All the links are from these older forum posts except that I have verified that they lead to the information indicated.

I also wondered are there any other sites other then the Nexus and the UESP Wiki for mods and other Morrowind utilities. As I continue to find other websites, tutorials and, Utilities I will list them here.

== Mod Utilities
=== Utilities by various authors:
* [[http://mw.modhistory.com/download-95-5496 | BSA Browser]] browse, search, pack, unpack, and register Bethesda Softworks Archive (bsa) files with a user friendly interface.
* BSA Registration Utility -It does the same thing like the old BSAReg, but KINGPIX has released the BSA Registration Utility with some interesting features.
  . [[http://forums.bethsoft.com/topic/1366245-relz-bsa-registration-utility/ | BSA Registration Utility]] Please note that downloading requires registration.
  . [[http://www.theassimilationlab.com/forums/files/file/1059-kingpixs-useful-tools/ | BSA Registration Utility]] TAL.
  . [[http://mw.modhistory.com/download-95-12389 | BSA Registration Utility]] MMH.
* [[https://www.gamefront.com/games/elite-force-2/file/dds-converter-1#Download | DDS Converter]]
* [[http://mw.modhistory.com/download-95-1662 | Enchanted Editor]] Is an advanced editor and has a lot of features that other tools doesn't have, but it's old and hard to understand how it works.
* [[http://mw.modhistory.com/download-95-14737 | escog]] This tool is used when you want to get rid of both evil GMST's or harmless GMST's and it's recommended to remove GMST's, but always check the readme first because some mods is cleaned even with GMST's.
* [[http://wryemusings.com/GMST%20Vaccine%200-02.zip | GMST Vaccine by Wrye]]
* [[https://www.nexusmods.com/morrowind/mods/43907/ | Groundcover Generation Package]] by Noteless
* [[https://github.com/loot/loot/releases | LOOT]] As of v14.6 LOOT Now supports Morrowind
* [[http://code.google.com/p/mlox/wiki/Mlox | mlox]] Is similar to what BOSS is for Oblivion to sort your loadorder.
* [[http://mw.modhistory.com/download-95-5265 | Morrowind Plugin Manager]] Is very easy to use for both installation/uninstallation of mods and it also supports mp3 files, but it doesn't support .7z archives and Wrye Mash has support for both 7z, MP3-files
* [[http://pmm.planetoblivion.de/Portal/index.php?option=com_content&task=view&id=263&Itemid=104 | NetImmerse Block Editor - NIBLE]]
* TESFaith Is a tool that you use for moving exterior cells from one cell to another cell.
  . [[http://mw.modhistory.com/download-95-2316 | TESFaith 0.95]] It's a tool that you use for moving exterior cells from one cell to another cell.
  . [[http://mw.modhistory.com/download-95-14908 | TESFaith 0.97alpha]] It's a tool that you use for moving exterior cells from one cell to another cell.
* [[http://mw.modhistory.com/download-95-5289 | The Elder Scrolls Advanced Mod Editor]] Is a powerful cleaning tool for modders similar to TES4Edit as a cleaning tool, who use TES Advanced Mod Editor before they release a mod or a patch.
* TES Mod Utility is another tool similar to TESAME, but I don't know if anyone are actually using it.
  . Obsolete according to the [[https://en.uesp.net/wiki/Tes3Mod:Modding#Outdated | Tes3Mod:Modding UESP Wiki]]
  * [[http://mw.modhistory.com/download-95-14568 | TES Mod Utility]]
  * [[http://mw.modhistory.com/download-95-5223 | TES Mod Utility TES Mod Utility 2.01]]
* [[https://www.nexusmods.com/morrowind/mods/3874 | TES Plugin Conflict Detector]] IIRC you can use TESPCD to clean mods, but it's an excellent tool for troubleshooting similar to what TES4Edit is for Oblivion.
* [[http://mw.modhistory.com/download-95-5258 | TES Dependency Tool Kit]] You can use it if you need to either add or remove a dependency for a mod for example an [[http://forums.bethsoft.com/topic/1376740-relz-morrowind-comes-alive-70/ | updated patch]] between two [[http://forums.bethsoft.com/topic/1380688-relz-tamriel-rebuilt-sacred-east-10-2/ | mods]]
* Wrye Mash
  . [[https://github.com/Wrye-Code-Collection/Wrye-Mash/releases/tag/88 | Wrye Mash Stand Alone v88]], by [[https://sharlikran.github.io/ | Sharlikran]]
  . [[https://www.nexusmods.com/morrowind/mods/45439 | Wrye Mash - Polemos fork - 2019]], by [[https://github.com/polemion/Wrye-Mash-Polemos | Polemos]]

=== Ghostwheel's Morrowind utilities and documents:
* [[http://mw.modhistory.com/download-95-14570 | BSAPACK Utility (with source code)]] – command line BSA archive packer/unpacker
* [[http://mw.modhistory.com/download-95-14571 | BSA Registration Utility]] provides automatic registration of the BSA archives (if you're too lazy to edit morrowind.ini)
* [[http://mw.modhistory.com/download-95-2140 | TES Rename 1.05]] powerful model/icon/texture file rename utility and more :)
* [[http://mw.modhistory.com/download-95-5249 | NIF Duplicator 1.0]] small helper utility for modders – can automatically duplicate given NIF for several texture files – just read documentation
* [[http://mw.modhistory.com/download-95-5267 | NIF Color 1.0]] small helper utility that can quickly update ambient/diffuse/specular color light values for all materials in multiple NIF files
* TES Plugin Tool 1.3 - Is a helper utility for the The Elder Scrolls: Morrowind game. ESM/ESP cleaning tool, leveled list merger, **object and dialog entry merger** and more.
  . [[https://wrye-code-collection.github.io/wcc-wiki/14-testoolghostwheel.html | TES Plugin Tool (TESTool) 1.3 Documentation]]
  . [[http://mw.modhistory.com/download-95-15255 | TES Plugin Tool 1.3]] - Slightly different CRC accoring to Dragon32
  . [[http://mw.modhistory.com/download-95-5234 | TES Plugin Tool 1.3]]

All programs are ZIP archives with text/html file documentation included in the archive. **READ IT!**
