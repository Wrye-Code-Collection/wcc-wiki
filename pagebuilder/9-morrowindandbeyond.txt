== Mod Utilities
=== Utilities by various authors:
* [[http://mw.modhistory.com/download-95-5496 | BSA Browser]] It's a tool similar to the BSA Commander for Oblivion, but it has a lot of other features listed below.
  * [[http://mw.modhistory.com/download-95-5215 | bsapack]] It is what it is to pack mod files into a BSA.
  * [[http://mw.modhistory.com/download-95-5212 | BSAReg]] Register any BSA file that exist in the Data Files folder into GetMWRoot - Search in Windows register after the path for the latest installation of Morrowind.
  * GetMWRoot - Search in Windows register after the path for the latest installation of Morrowind.
* BSA Registration Utility -It does the same thing like the old BSAReg, but KINGPIX has released the BSA Registration Utility with some interesting features.
  . [[http://forums.bethsoft.com/topic/1366245-relz-bsa-registration-utility/ | BSA Registration Utility]] Please note that downloading requires registration.
  . [[http://download.fliggerty.com/download-58-633 | BSA Registration Utility]] GHF.
  . [[http://www.theassimilationlab.com/forums/files/file/1059-kingpixs-useful-tools/ | BSA Registration Utility]] TAL.
  . [[http://mw.modhistory.com/download-95-12389 | BSA Registration Utility]] MMH.
* [[https://github.com/loot/loot/releases | LOOT]] As of v14.6 LOOT Now supports Morrowind
* [[http://code.google.com/p/mlox/wiki/Mlox | mlox]] Is similar to what BOSS is for Oblivion to sort your loadorder.
* [[http://mw.yacoby.net/Mods/escog.htm | escog]] This tool is used when you want to get rid of both evil GMST's or harmless GMST's and it's recommended to remove GMST's, but always check the readme first because some mods is cleaned even with GMST's.
* [[http://mw.modhistory.com/download-95-1662 | Enchanted Editor]] Is an advanced editor and has a lot of features that other tools doesn't have, but it's old and hard to understand how it works.
* [[http://mw.modhistory.com/download--2316 | TESFaith]] It's a tool that you use for moving exterior cells from one cell to another cell.
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/ | TESTool]] Is most basic tool there is and you must use TESTool to clean mods with merged objects, but remember sometimes TESTool can create more problems than fixing them so always backup your files.
* [[http://mw.modhistory.com/download-95-5289 | The Elder Scrolls Advanced Mod Editor]] Is a powerful cleaning tool for modders similar to TES4Edit as a cleaning tool, who use TES Advanced Mod Editor before they release a mod or a patch.
* TES Mod Utility is another tool similar to TESAME, but I don't know if anyone are actually using it.
  . Obsolete according to the [[https://en.uesp.net/wiki/Tes3Mod:Modding#Outdated | Tes3Mod:Modding UESP Wiki]]
  * [[http://mw.modhistory.com/download-95-14568 | TES Mod Utility]]
  * [[http://mw.modhistory.com/download-95-5223 | TES Mod Utility TES Mod Utility 2.01]]
* [[https://www.nexusmods.com/morrowind/mods/3874 | TES Plugin Conflict Detector]] IIRC you can use TESPCD to clean mods, but it's an excellent tool for troubleshooting similar to what TES4Edit is for Oblivion.
* [[http://mw.modhistory.com/download--5258 | TES Dependency Tool Kit]] You can use it if you need to either add or remove a dependency for a mod for example an [[http://forums.bethsoft.com/topic/1376740-relz-morrowind-comes-alive-70/ | updated patch]] between two [[http://forums.bethsoft.com/topic/1380688-relz-tamriel-rebuilt-sacred-east-10-2/ | mods]]
* [[http://mw.modhistory.com/download-95-5265 | Morrowind Plugin Manager]] Is very easy to use for both installation/uninstallation of mods and it also support mp3 files, but it doesn't support .7z archives and Wrye Nash have support for both 7z, MP3-files (only the) [[http://forums.bethsoft.com/topic/1331996-yacobys-wrye-mash-fork/ | latest]]
* Wrye Mash
  . [[https://github.com/Wrye-Code-Collection/Wrye-Mash/releases/tag/88 | Wrye Mash Stand Alone v88]], by [[https://sharlikran.github.io/ | Sharlikran]]
  . [[https://www.nexusmods.com/morrowind/mods/45439 | Wrye Mash - Polemos fork - 2019]], by [[https://github.com/polemion/Wrye-Mash-Polemos | Polemos]]

=== Ghostwheel's Morrowind utilities and documents:
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/TES3_BSA_File_Format.htm | TES3 BSA File Format Description]]
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/bsapack.zip | BSAPACK Utility (with source code)]] – command line BSA archive packer/unpacker
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/bsareg.zip | BSA Registration Utility]] provides automatic registration of the BSA archives (if you're too lazy to edit morrowind.ini)
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/TESRename.zip | TES Rename 1.05]] powerful model/icon/texture file rename utility and more :)
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/NIFDuplicator.zip | NIF Duplicator 1.0]] small helper utility for modders – can automatically duplicate given NIF for several texture files – just read documentation
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/NIFColor.zip | NIF Color 1.0]] small helper utility that can quickly update ambient/diffuse/specular color light values for all materials in multiple NIF files
* [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/TESTool.zip | TES Plugin Tool 1.3]] – ESM/ESP cleaning tool, leveled list merger, **object and dialog entry merger** and more. Documentation can be found [[http://web.archive.org/web/20040428231923/http://www34.brinkster.com/ghostwheel/TESTool.htm | HERE]].

All programs are ZIP archives with text/html file documentation included in the archive. **READ IT!**
